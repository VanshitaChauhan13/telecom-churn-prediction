Telecom Customer Churn Prediction
This repository features a Jupyter Notebook for predicting customer churn in telecom using the Telco dataset from Kaggle. It includes EDA, preprocessing (Label Encoding), model comparisons (Random Forest, SVM), and evaluation metrics. The top model (Random Forest) achieves ~78% accuracy.

Quick Setup
Clone: git clone https://github.com/yourusername/telecom-churn-prediction.git
Install: pip install -r requirements.txt
Run: jupyter notebook telecom_churn.ipynb

Key Files
telecom_churn.ipynb: Core analysis.
Telco-Customer-Churn.csv: Dataset.
rf_churn_model.pkl: Trained model.
requirements.txt: Dependencies.
